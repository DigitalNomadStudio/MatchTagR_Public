# MatchTagR Screenshots

Add your app screenshots here for the website carousel.

## Required Screenshots

Place the following 6 screenshots in this directory:

1. **screenshot1.png** - Home Screen (showing the three mode cards)
2. **screenshot2.png** - Live Tag Match mode in action
3. **screenshot3.png** - Tag Uploaded Video mode
4. **screenshot4.png** - Record & Tag Match mode
5. **screenshot5.png** - Event Timeline/List view
6. **screenshot6.png** - Highlight Reels or Settings screen

## Image Specifications

- **Format:** PNG (recommended) or JPG
- **Dimensions:** iPhone screenshot dimensions (e.g., 1170 x 2532 for iPhone 13 Pro)
- **Aspect Ratio:** Maintain iPhone aspect ratio (approximately 9:19.5)
- **File Size:** Optimize for web (under 500KB each recommended)

## Tips for Great Screenshots

- Use the iOS Simulator or a real device
- Capture screens that showcase key features
- Ensure good contrast and readability
- Show the app in use with realistic data
- Consider using a tool like [Previewed](https://previewed.app/) or [Screenshot.rocks](https://screenshot.rocks/) for polished mockups

## How to Export from Xcode Simulator

1. Run your app in the iOS Simulator
2. Navigate to the screen you want to capture
3. Press `Cmd + S` to save a screenshot
4. The screenshot will be saved to your Desktop
5. Rename and move it to this directory

## Testing

Once you add the screenshots, the carousel will automatically display them. If a screenshot is missing, a placeholder will be shown instead.
